# .agent/skills/

This directory is reserved for Claude Code **maintainer-side** skills —
workflows used while developing unified-live itself. It is currently
empty by design.

User-facing AI documentation lives elsewhere in this repo:

- `llms.txt` / `llms-full.txt` / `llms-small.txt` at
  https://sugar-cat7.github.io/unified-live/ (generated by
  `starlight-llms-txt` during the docs build)
- Claude Code plugin marketplace at `.claude-plugin/marketplace.json`,
  with the plugin itself under `plugins/unified-live/`
- Context7 MCP, registered via `context7.json` at the repo root

If you add maintainer-only skills here, prefix them clearly and document
them in `CLAUDE.md` / `AGENTS.md` under "Claude Code Operations".
