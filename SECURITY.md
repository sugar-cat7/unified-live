# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in unified-live, please report it through [GitHub's Private Vulnerability Reporting](https://github.com/sugar-cat7/unified-live/security/advisories/new).

**Please do not report security vulnerabilities through public issues.**

## What to Expect

- We will acknowledge receipt within 7 days.
- We will provide an initial assessment within 14 days.
- We will work with you to understand and address the issue before any public disclosure.

## Scope

This policy applies to all packages in the `@unified-live` scope:

- `@unified-live/core`
- `@unified-live/youtube`
- `@unified-live/twitch`
- `@unified-live/twitcasting`
